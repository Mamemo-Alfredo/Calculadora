# 🧮 Calculadora Web

Aplicação de calculadora desenvolvida com **HTML, CSS e JavaScript**, com suporte às operações matemáticas básicas e foco em interatividade e experiência do usuário.

---

## 📌 Sobre o Projeto

Este projeto consiste em uma calculadora funcional que permite realizar operações matemáticas diretamente no navegador.
A aplicação foi construída com uma interface simples, responsiva e intuitiva.

---

## 🚀 Tecnologias Utilizadas

* 🌐 HTML5
* 🎨 CSS3
* ⚡ JavaScript

---

## 🎯 Funcionalidades

* ➕ Adição
* ➖ Subtração
* ✖️ Multiplicação
* ➗ Divisão
* 🧹 Limpar visor
* ⌫ Apagar último número
* 📱 Layout responsivo

---

## ⚙️ Como Funciona

* O usuário insere números e operadores através dos botões
* As operações são exibidas em tempo real no visor
* O resultado é calculado ao pressionar "="
* A lógica é controlada via JavaScript
